# Guide parent

Suivez la scolarité de votre enfant et payez en ligne, depuis votre téléphone.

## 1. Créer votre compte
1. Ouvrez l'application GesSchool dans votre navigateur.
2. Onglet **Inscription** → votre e-mail + un mot de passe → **Créer le compte**.

## 2. Rejoindre votre enfant
1. Demandez à l'école votre **code de liaison** (8 caractères, ex. `3F9A2B7C`).
2. À l'écran d'accueil, choisissez **👪 Je suis un parent**.
3. Saisissez le code → **Accéder à l'espace parent**.

> Un seul code suffit même si vous avez plusieurs enfants dans l'école.

## 3. Consulter
Pour chaque enfant, des onglets : **Notes**, **Emploi du temps**, **Fournitures**, **Paiements**, **Absences**. Tout est mis à jour en temps réel par l'école.

## 4. Payer une facture (mobile money)
1. Onglet **Paiements** → bouton **Payer** sur la facture concernée.
2. Choisissez **Wave / Orange Money / Free Money** → le numéro de l'école s'affiche.
3. Payez depuis votre application mobile money, en indiquant le **numéro de facture** en référence.
4. Revenez et **Déclarez le paiement** (montant + réf. de la transaction).
5. L'école valide : la facture passe à « payée ».

## 5. Notifications
Activez les **notifications** pour être prévenu des nouvelles notes, absences, factures et rappels de paiement. Sur téléphone, « Ajouter à l'écran d'accueil » installe l'app comme une appli classique.
