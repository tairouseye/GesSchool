Démarrer en 30 minutes

Le strict nécessaire pour rendre votre école opérationnelle sur GesSchool.

Étape 1 — Créer l'école (5 min)

Inscription → Je gère une école → assistant en 3 étapes : identité (nom, sigle, logo), cycles ouverts, responsable + année scolaire (trimestres ou semestres). L'année et les périodes sont créées automatiquement.

Étape 2 — Construire la structure (10 min) · menu *Structure*

  1. Ajoutez vos niveaux (6e, CP, Seconde…).
  2. Générez les classes en lot (base du nom + nombre + suffixe A/B/C).
  3. Ajoutez les matières (avec code).
  4. Lycée : créez les séries puis la grille de coefficients.

Étape 3 — Inscrire les élèves (10 min) · menu *Élèves*

  • + Nouvel élève (un par un), ou ↑ Importer (Excel) pour toute une liste.
  • Astuce import : la colonne Classe doit correspondre exactement au libellé d'une classe créée à l'étape 2.

Étape 4 — Préparer la facturation (5 min) · menu *Paiements*

  1. Onglet Grille tarifaire : créez vos frais (Inscription, Scolarité…), cochez « obligatoire ».
  2. Onglet Paiement mobile : saisissez vos numéros Wave / OM / Free Money.
  3. ⚡ Générer en lot : une facture par élève d'un niveau, en un clic.

Ensuite, au quotidien

  • Notes puis Bulletins (impression PDF en 1 clic).
  • Recouvrement : relances automatiques des impayés + rappels push aux parents.
  • Espace parent : générez un Code parent depuis la fiche élève et donnez-le aux familles.

Astuce : activez seulement les modules dont vous avez besoin (menu *Paramètres → Modules actifs*).
